# 1. 快速开始

## 1.1. 定义

统计学（英文 Statistics），是一门研究如何收集、分析、解释和呈现数据的学科。

统计学利用数学和概率理论来研究数据的规律性和变异性，帮助人们更好地理解数据背后的信息和趋势。统计学包括描述统计（对数据进行总结和展示）、推断统计（通过样本推断总体特征）、概率统计（基于概率理论进行推断和预测）等内容。
